# Ravindra Bhati — Modernized Portfolio

This branch modernize-portfolio contains an initial Next.js + TypeScript + Tailwind scaffold and migrated content from the original static site.

How to run locally:

1. git clone git@github.com:RavindraBhati7/RavindraBhati7.github.io.git
2. cd RavindraBhati7.github.io
3. git checkout modernize-portfolio
4. npm install
5. npm run dev

Notes:
- This is an initial scaffold. Replace placeholder assets in /public and refine animations and styling as needed.
- Deployment: Vercel is recommended. Connect the repository and use the modernize-portfolio branch or merge to main.
